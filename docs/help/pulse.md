# Pulse - Operational Metrics

Pulse tracks real-time operational metrics and detects anomalies.

## Key Concepts
- **Metrics**: Quantitative measures of business performance.
- **Anomalies**: Statistically significant deviations from expected behavior.
- **Process Flows**: Visual representations of key business processes.

## Common Actions
- Monitor metric dashboards
- Investigate anomaly alerts
- Compare metrics across time periods
- Export metric data
