# Chat - AI Assistant

Chat provides a conversational interface to Atheon's AI.

## Key Concepts
- **Context-Aware Queries**: The AI understands your organization's data.
- **Citations**: Responses include references to data sources.
- **Conversation History**: Previous conversations are saved.

## Common Actions
- Ask questions about business performance
- Request analysis of specific metrics
- Generate reports through conversation
- Explore what-if scenarios
