# IAM - Identity & Access Management

IAM manages users, roles, and permissions.

## Key Concepts
- **Roles**: Predefined permission sets (Admin, Executive, Manager, Analyst, Operator).
- **HITL**: Human-in-the-Loop approval workflows.
- **Tenant Isolation**: Each organization's data is isolated.

## Common Actions
- Invite new users and assign roles
- Review and modify user permissions
- Audit user activity logs
- Configure HITL approval thresholds
