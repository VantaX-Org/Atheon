# Audit - Activity Log

The Audit page provides a comprehensive log of all actions.

## Key Concepts
- **Audit Trail**: Every action and event is recorded.
- **Anomaly Detection**: Unusual activity is flagged.
- **Compliance**: Logs support regulatory requirements.

## Common Actions
- Search and filter audit entries
- Export audit data to CSV
- Investigate flagged anomalies
- Review user activity patterns
