# Integrations - External Connections

Integrations manages connections to external systems.

## Key Concepts
- **Connectors**: Pre-built adapters for business systems (Xero, SAP, etc.).
- **Sync Schedule**: Configurable data synchronization frequency.
- **Data Mapping**: Field-level mapping between systems.

## Common Actions
- Add new integration connections
- Test connection health
- Configure sync schedules
- Review sync history and error logs
