# Mind - AI Configuration

Mind manages the AI models and prompt engineering powering Atheon.

## Key Concepts
- **Autonomy Tiers**: Levels of AI independence (T1-T5).
- **LLM Configuration**: Model parameter settings.
- **Prompt Playground**: Test queries against the AI engine.

## Common Actions
- Adjust model parameters
- Test prompts in the playground
- Review AI cost breakdowns
- Switch between LLM providers
