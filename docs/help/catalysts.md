# Catalysts Help

## Overview
Catalysts are automated business process agents that monitor, analyze, and act on your operational data.

## Clusters
A cluster groups related sub-catalysts by business domain (e.g., Finance, HR, Procurement).

### Creating a Cluster
1. Click **New Cluster** or use an industry template
2. Name the cluster and select a domain
3. Configure sub-catalysts within the cluster

## Sub-Catalysts
Each sub-catalyst handles a specific task:
- **Autonomy Tiers**: Read-Only, Assisted, Transactional
- **Schedules**: Configure how often the sub-catalyst runs
- **Data Sources**: Connect to ERP, CSV, API, or database sources

## Execution
- **Automatic**: Runs on schedule based on configured frequency
- **Manual**: Click "Run Now" to trigger immediate execution
- **Quick Run**: Streamlined execution with file upload support

## Analytics
View run history, KPI trends, success rates, and performance metrics for each sub-catalyst.
