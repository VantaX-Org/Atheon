# Dashboard Help

## Overview
The Atheon Dashboard provides a real-time overview of your organization's operational health across all dimensions.

## Health Score
The health score is a composite metric (0-100) calculated from:
- **Financial Health**: Revenue trends, cost efficiency, budget adherence
- **Operational Efficiency**: Process throughput, error rates, SLA compliance
- **Compliance & Governance**: Regulatory adherence, audit scores, policy compliance
- **Strategic Alignment**: Goal progress, initiative tracking, milestone completion
- **Technology Readiness**: System uptime, integration health, data quality

## KPI Cards
Each card shows a key metric with:
- Current value
- Trend indicator (up/down/stable)
- Sparkline showing recent history
- Click to drill down for details

## Quick Actions
- **Run Catalyst**: Trigger a manual catalyst execution
- **View Reports**: Access the latest board report
- **Manage Settings**: Configure dashboard preferences
