# Settings Help

## Profile
Update your display name, email, and avatar.

## Security
- **Password**: Change your password (must meet strength requirements)
- **MFA**: Enable/disable multi-factor authentication using TOTP
- **API Keys**: Manage API keys for programmatic access

## Notifications
Configure which events trigger notifications and how you receive them.

## Appearance
- **Theme**: Switch between light and dark mode
- **Language**: Select your preferred language (English, Afrikaans)

## Organization
Admin-only settings for tenant configuration, user management, and billing.
